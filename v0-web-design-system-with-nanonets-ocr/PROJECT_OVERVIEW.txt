================================================================================
                          USER JOURNEY
================================================================================

Path 1: AI-Powered Design (Using OCR)
┌─────────────────────────────────────────────────────────────────────────┐
│ 1. Land on home page (/)                                               │
│ 2. Click "Launch Editor"                                               │
│ 3. Enter Design Mode with editor open                                  │
│ 4. Click "Upload Image" button                                         │
│ 5. Select design mockup/screenshot                                     │
│ 6. Nanonets AI analyzes image                                          │
│ 7. Detected components appear on canvas                                │
│ 8. Click component to select                                           │
│ 9. Edit properties in right panel                                      │
│ 10. Switch to Export mode                                              │
│ 11. Click "Download HTML" or "Copy Code"                               │
│ 12. ✅ Done! Use the exported HTML in your project                     │
└─────────────────────────────────────────────────────────────────────────┘

Path 2: Manual Design (Drag-Drop)
┌─────────────────────────────────────────────────────────────────────────┐
│ 1. Land on home page (/)                                               │
│ 2. Click "Launch Editor"                                               │
│ 3. See Component Palette on the left                                   │
│ 4. Drag "Button" from palette to canvas                                │
│ 5. Drag "Text" from palette to canvas                                  │
│ 6. Drag "Image" from palette to canvas                                 │
│ 7. Repeat until design complete                                        │
│ 8. Click each component to select                                      │
│ 9. Edit text, colors, size in properties panel                         │
│ 10. Arrange components on canvas                                       │
│ 11. Switch to Export mode                                              │
│ 12. Download or copy generated HTML                                    │
│ 13. ✅ Done! Use the HTML in your project                              │
└─────────────────────────────────────────────────────────────────────────┘
