📝 USAGE IN CODE
================================================================================

Import API Client:
  import apiClient from '@/lib/api-client';

Analyze Image:
  const result = await apiClient.ocr.analyzeImage(file);

Create Project:
  const proj = await apiClient.projects.create({
    name: 'My Project',
    description: 'Test'
  });

Get Projects:
  const projects = await apiClient.projects.list();

Update Project:
  await apiClient.projects.update(projectId, {
    name: 'Updated Name'
  });

Export to HTML:
  const exported = await apiClient.export.toHTML(projectId);
  console.log(exported.html);

================================================================================
📞 TROUBLESHOOTING
================================================================================

"Mock API won't start"
  → Check port 3001 is not in use
  → Kill any existing process: lsof -i :3001

"CORS errors"
  → Mock API handles CORS automatically
  → Verify NEXT_PUBLIC_API_BASE_URL is correct
  → Restart both services

"API not responding"
  → Check mock API is running
  → Verify .env.local has correct URL
  → Check Network tab in DevTools

"Schema syntax error"
  → Validate with: pnpm exec prism mock schema.yaml --validate
