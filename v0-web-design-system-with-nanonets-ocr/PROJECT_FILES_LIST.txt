How to Download
================

Option 1: Download ZIP
- Click 3 dots (...) in v0 top right
- Select "Download ZIP"
- Extract with: unzip, 7-Zip, or WinRAR

Option 2: Connect GitHub
- Click GitHub logo in sidebar
- Link to your repository
- Clone locally: git clone <url>

After Download
==============

1. Extract/clone the project
2. cd your-project-name
3. npm install
4. cp .env.example .env.local
5. npm run mock-api (Terminal 1)
6. npm run dev (Terminal 2)
7. Visit http://localhost:3000
