HOW TO USE
================================================================================

1. LANDING PAGE
   - Visit http://localhost:3000
   - See features and how it works
   - Click "Launch Editor" button

2. DESIGN EDITOR
   - Left: Component palette (drag components)
   - Center: Canvas (main design area)
   - Right: Properties panel (edit selected component)
   
3. ADD COMPONENTS
   - Click components in palette
   - Or drag-drop onto canvas
   - Edit properties on right panel

4. UPLOAD IMAGE
   - Click "Upload Image" button
   - Select image file
   - AI detects components (uses mock API)
   - Components auto-added to canvas

5. EXPORT HTML
   - Click "Export" button
   - Choose: Preview, Code, or Download
   - Get production-ready HTML

================================================================================
TROUBLESHOOTING
================================================================================

Issue: Only see SCHEMA_DELIVERY.txt in ZIP
→ Solution: All files are there. Extract with proper tool.
→ See: IF_FILES_MISSING.md

Issue: npm install fails
→ Solution: rm -rf node_modules && npm install

Issue: Port 3000 already in use
→ Solution: npm run dev -- -p 3001

Issue: Mock API won't start
→ Solution: npm install -g @stoplight/prism-cli

Issue: Files not showing in editor
→ Solution: Open full project folder (not just root)

For more help:
→ Read: SETUP.md or DOWNLOAD_GUIDE.md
